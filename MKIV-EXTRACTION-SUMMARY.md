### 3. Closing Straps (Cintas de Fecho) Specifications

#### Material & Type
- **Material**: Stainless steel metal straps
- **Securing Method**: Crimp (not screw/bolt)
- **Heat Shrink**: Optional heat-shrink sleeve available for encapsulation

#### Installation Process
1. Position strap ends ~25mm above container rim
2. Apply tensioning tool at midpoint across container rims
3. Tension until tool base rests in lower rim
4. Secure with crimp using crimping tool
5. **NO TORQUE SPECIFICATION** - uses tensioning tool method instead

#### Quantity by Container
- **Small containers** (4p-10p): 2-4 straps
- **Medium containers** (12p-16p Throwover): 4-8 straps
- **Large containers** (20p-25p Throwover): 8-10 straps
- **Davit-launch**: +2-4 additional straps compared to throwover
- **Davit-launch special**: Always includes **2 hand loop straps** for davit ring access door
- **High stowage** (>18m): Additional strap at each container end

#### Safety Requirements
⚠️ **Critical Safety Notes:**
- Operator MUST stand to one side of strap
- Proper clothing and eye protection REQUIRED
- Crimps MUST be on opposite side to roll direction
- Too much tension will break strap (injury risk)

## Important Observations

### Missing Data
1. **15p capacity**: Not found in manual - capacities jump from 12p to 16p
2. **Strap torque**: No torque specification exists - crimping method used instead
3. **Some container dimensions**: Certain B-pack and davit-launch configurations don't have dimensions specified in summary tables

### Container Variations
- Each capacity has **multiple container options** (MK 10, MK 14, MK 16, MK 18, 4N, G21)
- Container choice depends on:
  - SOLAS pack type (A-pack vs B-pack)
  - Stowage height (18m vs 36m)
  - Launch method (throwover vs davit-launch)
  - Space constraints (cylindrical vs flatpack)

### SOLAS Pack Types
- **A-Pack**: Full equipment - heavier (+20-30 kg)
- **B-Pack**: Reduced equipment - lighter

### Cylinder Alternative Models
- For 16p throwover: Two cylinder options exist (8.80/0.44 kg or 6.85/0.212 kg)
- All other capacities: Single standard configuration
